How It's Built: The Backend Engine

The Power of a Custom API

A Tableau Extension is more than just a front-end widget; its true power comes from the backend API that does the heavy lifting. We work with your team to build a robust, scalable API that connects your Tableau dashboard to any service you need.

  • Connect to Data Warehouses: Execute complex queries in Snowflake, BigQuery, and more.
  • Integrate GenAI Models: Send data to models like Gemini, Claude, or Llama for analysis.
  • Trigger Actions: Start data pipelines, send emails, or update other systems.
  • Enforce Business Logic: Run complex calculations that are too difficult or slow to perform in Tableau itself.
from fastapi import FastAPI
from pydantic import BaseModel
import snowflake.connector as snow
from vertexai.generative_models import GenerativeModel

app = FastAPI()

class InputModel(BaseModel):
    data: str
    intent: str

@app.post("/process_persona")
def process_persona(input_data: InputModel):
    # 1. Parse data from Tableau
    data_df = parse_tableau_data_as_dataframe(
        data=input_data.data
    )

    # 2. Craft a prompt for the AI
    prompt = f"User Intention: {input_data.intent}\nData: {data_df.to_markdown()}"

    # 3. Generate insights using GenAI
    model = GenerativeModel('gemini-1.5-flash-002')
    response = model.generate_content([prompt])

    # 4. Return summary to Tableau
    return {"response": response.text}
